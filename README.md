Management Inventoty and Customer

Este proyecto es una aplicación para la gestión de inventario y clientes en un almacen. Permite registrar, actualizar y eliminar productos y clientes, así como generar facturas y automatizar la salida de inventario.


Funcionalidades clave:

Registro y gestión de clientes.
Registro y gestión de productos.
Generación de facturas.
Control de acceso mediante roles (administrador y usuario).
Integración con base de datos MySQL.
Interfaz gráfica utilizando Java Swing.
Seguridad mediante encriptación SHA-256.

Instalacion

git clone https://github.com/pegaso02/ManagementInventotyCustomer.git
cd ManagementInventotyCustomer


Tecnologías Utilizadas


Java
JavaFX/Swing
MySQL
